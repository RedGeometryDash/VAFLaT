About VAFLaT

VAFLaT (pronounced V.A. flat) stands for Visual Addon for Language Transfer. Language Transfer is completely audio-based which is fine for many but others also need a visual component to interact with in order to learn optimally. VAFLaT attempts to add that visual component without implementing memorization.

VAFLaT offers three types of practice questions for each lesson, but of course there are usually more than three questions per lesson. Those three question types are: multiple choice, fill in the blank, and card matching.

VAFLaT is a school project and is NOT an official Language Transfer product. We only use Language Transfer's audio.
